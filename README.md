# xpar - Проект для анализа графем в строках

## Описание проекта

Проект xpar представляет собой консольное приложение на C#, предназначенное для анализа и отображения информации о графемах в строках текста. Приложение читает строки из стандартного ввода, вычисляет ширину каждой слова по правилам Unicode (используя библиотеку Wcwidth), и выводит результаты анализа.

```
xpar/

└── bin/                # Директория для скомпилированных файлов
    └── Debug/

```

## Основные компоненты

### Program.cs
- Основной класс `Program` с методом `Main`
- Логика чтения строк из консоли и их обработки
- Метод `Transform` для анализа строки
- Метод `CountGraphemes` для подсчета графем в словах

### xpar.csproj
- Файл проекта .NET SDK
- Целевая платформа: .NET 5.0
- Ссылка на пакет Wcwidth версии 2.0.0 для работы с шириной символов Unicode

## Как использовать

1. Скомпилируйте проект:
   ```
   dotnet build
   ```

2. Запустите приложение:
   ```
   dotnet run
   ```

3. Введите строки текста и нажмите Enter для получения анализа графем.

Пример вывода:
```
Hello World[5, 5]
```

## Зависимости

- .NET 5.0 Runtime
- Библиотека Wcwidth для корректного расчета ширины символов Unicode

## Особенности реализации

Приложение использует:
- `StringInfo.GetTextElementEnumerator()` для правильной обработки графем
- `Wcwidth.UnicodeCalculator.GetWidth()` для получения ширины символов в терминах
- Нормализацию текста по форме KD (Normalization Form KD)
