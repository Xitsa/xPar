# Шаг 3: Подключение парсинга аргументов в Main

## Цель шага

Обновить `Program.Main` для чтения переменных окружения и передачи аргументов командной строки в `ParOptions.Parse`.

## План шага

### 1. Инициализация charset'ов из переменных окружения

В `Main()` создать charset'ы из переменных окружения (как в оригинальном par.c):

```csharp
// Из переменных окружения (если не заданы — пустые строки)
var bodyChars = Charset.Parse(Environment.GetEnvironmentVariable("PARBODY") ?? "");
var protectChars = Charset.Parse(Environment.GetEnvironmentVariable("PARPROTECT") ?? "");
var quoteChars = Charset.Parse(Environment.GetEnvironmentVariable("PARQUOTE") ?? "> ");

// Значения по умолчанию (без переменных окружения)
var whiteChars = Charset.Parse(" \f\n\r\t\v");
var terminalChars = Charset.Parse(".?!:");
```

### 2. Обработка PARINIT

Если переменная окружения `PARINIT` задана, разобрать её как аргументы командной строки:

- Аргументы в `PARINIT` разделяются начальным набором whitespace (до обработки опций W)
- Аргументы из PARINIT обрабатываются **перед** аргументами командной строки
- PARINIT имеет приоритет (аргументы командной строки могут переопределять)

**Значение PARINIT по умолчанию:**

Согласно par.doc (Quick Start), рекомендуемое значение:
```
rTbgqR B=.,?'_A_a_@ Q=_s>|
```

НО: в оригинальном par PARINIT **не задаётся по умолчанию** — если переменная не установлена, она просто не обрабатывается.
Значение из Quick Start — это рекомендация для пользователя, а не дефолт программы.

**Решение для xpar:**
- Если `PARINIT` не задан — НЕ устанавливаем его автоматически (следуем оригиналу)
- Документируем рекомендуемое значение в README

### 3. Разбор аргументов и обработка help/version

Логика в `Main`:

1. Инициализировать charset'ы из переменных окружения
2. Если `PARINIT` задан — разобрать его и применить к charset'ам и параметрам
3. Разобрать аргументы командной строки (могут переопределять PARINIT)
4. Если `Help` — вывести справку и выйти
5. Если `Version` — вывести версию и выйти
6. Иначе — прочитать stdin и вызвать `Transform`

### 4. Обновить Program.cs

```csharp
static void Main(string[] args)
{
    Console.InputEncoding = Encoding.UTF8;
    Console.OutputEncoding = Encoding.UTF8;

    // 1. Инициализация charset'ов из переменных окружения
    var bodyChars = Charset.Parse(Environment.GetEnvironmentVariable("PARBODY") ?? "");
    var protectChars = Charset.Parse(Environment.GetEnvironmentVariable("PARPROTECT") ?? "");
    var quoteChars = Charset.Parse(Environment.GetEnvironmentVariable("PARQUOTE") ?? "> ");
    var whiteChars = Charset.Parse(" \f\n\r\t\v");
    var terminalChars = Charset.Parse(".?!:");

    // 2. Обработка PARINIT (если задан)
    string[] allArgs;
    var parinit = Environment.GetEnvironmentVariable("PARINIT");
    if (!string.IsNullOrEmpty(parinit))
    {
        // Разделяем по начальным whitespace (до обработки W)
        var initArgs = parinit.Split(new[] { ' ', '\f', '\n', '\r', '\t', '\v' },
            StringSplitOptions.RemoveEmptyEntries);
        // Сначала PARINIT, потом командная строка
        allArgs = initArgs.Concat(args).ToArray();
    }
    else
    {
        allArgs = args;
    }

    // 3. Разбор всех аргументов
    var options = ParOptions.Parse(allArgs, bodyChars, protectChars, quoteChars, whiteChars, terminalChars);

    // 4. Обработка help
    if (options.Help)
    {
        PrintHelp();
        return;
    }

    // 5. Обработка version
    if (options.Version)
    {
        Console.WriteLine("xpar 1.0.0");
        return;
    }

    // 6. Чтение stdin и трансформация
    var transformer = new StringTransformer();
    var lines = new List<string>();
    string? curLine;

    do
    {
        curLine = Console.ReadLine();
        if (curLine != null)
        {
            lines.Add(curLine);
        }
    }
    while (curLine != null);

    var results = transformer.Transform(lines);
    foreach (var result in results)
    {
        Console.WriteLine(result);
    }
}
```
