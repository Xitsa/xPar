# Шаг 15: Построение выходных строк (Construct the lines)

## Цель шага

Реализовать метод `ConstructLines` в `ReformatModule` — финальный этап форматирования,
который берёт слова с расставленными разрывами строк (NextLine) и строит
готовые выходные строки с префиксами, телом и суффиксами.

## Контекст

В оригинале блок "Construct the lines" (reformat.c строки 459–522) — это часть
функции `reformat()`, которая:

1. Итерирует по строкам абзаца (включая дополнительные строки до `hang`).
2. Для каждой строки:
   a. Вычисляет `numgaps` (количество промежутков) и `extra` (свободное место).
   b. Определяет `linelen` (общую длину строки с префиксом и суффиксом).
   c. Копирует **префикс** из исходных строк или fallback.
   d. Копирует **тело** (слова) с распределением дополнительных пробелов (justification).
   e. Заполняет пробелами до нужной длины.
   f. Копирует **суффикс** из исходных строк или fallback.
3. Добавляет завершающую строку (null-терминатор в оригинале).

Перед этим блоком есть фаза **touch** (reformat.c строки 445–455):
```c
if (!just && touch) {
    L = 0;
    w1 = head->next;
    while (w1) {
        for (linelen = w1->length, w2 = w1->next;
             w2 != w1->nextline;
             linelen += 1 + isshifted(w2) + w2->length, w2 = w2->next);
        if (linelen > L) L = linelen;
        w1 = w2;
    }
}
```
Она изменяет L на фактическую длину самой длинной строки (при `touch=1` и `just=0`).

## Оригинал (reformat.c строки 459–522)

```c
/* Construct the lines: */
pbuf = newbuffer(sizeof (char *), errmsg);
if (*errmsg) goto rfcleanup;

numout = 0;
w1 = head->next;
while (numout < hang || w1) {
    if (w1)
        for (w2 = w1->next, numgaps = 0, extra = L - w1->length;
             w2 != w1->nextline;
             ++numgaps, extra -= 1 + isshifted(w2) + w2->length, w2 = w2->next);
    linelen = suffix || (just && (w2 || last)) ?
                L + affix :
                w1 ? prefix + L - extra : prefix;
    q1 = malloc((linelen + 1) * sizeof (char));
    ...
    additem(pbuf, &q1, errmsg);
    ++numout;
    q2 = q1 + prefix;

    /* Копирование префикса */
    if      (numout <= numin) memcpy(q1, inlines[numout - 1], prefix);
    else if (numin  >  hang ) memcpy(q1, endline[-1],         prefix);
    else {
        if (afp > prefix) afp = prefix;
        memcpy(q1, endline[-1], afp);
        q1 += afp;
        while (q1 < q2) *q1++ = ' ';
    }

    q1 = q2;
    if (w1) {
        phase = numgaps / 2;
        for (w2 = w1;  ;  ) {
            memcpy(q1, w2->chrs, w2->length);
            q1 += w2->length;
            w2 = w2->next;
            if (w2 == w1->nextline) break;
            *q1++ = ' ';
            if (just && (w1->nextline || last)) {
                phase += extra;
                while (phase >= numgaps) {
                    *q1++ = ' ';
                    phase -= numgaps;
                }
            }
            if (isshifted(w2)) *q1++ = ' ';
        }
    }

    q2 += linelen - affix;
    while (q1 < q2) *q1++ = ' ';

    q2 = q1 + suffix;
    /* Копирование суффикса */
    if      (numout <= numin) memcpy(q1, suffixes[numout - 1], suffix);
    else if (numin  >  hang ) memcpy(q1, suffixes[numin  - 1], suffix);
    else {
        if (fs > suffix) fs = suffix;
        memcpy(q1, suffixes[numin - 1], fs);
        q1 += fs;
        while(q1 < q2) *q1++ = ' ';
    }
    *q2 = '\0';
    if (w1) w1 = w1->nextline;
}
```

## Алгоритм (пошагово)

### Цикл построения строк

```
while (numout < hang || w1 != null):
```
Продолжаем, пока не вывели хотя бы `hang` строк И не обработали все слова.

### Шаг 1: Вычисление numgaps и extra

Для текущей строки (начиная с `w1`):
```
extra = L - w1.Width
numgaps = 0
for w2 = words[w1.NextLine]:
    numgaps++
    extra -= 1 + (w2.Shifted ? 1 : 0) + w2.Width
```

Если `w1 == null` (дополнительная строка после hang): `numgaps = 0, extra = L`.

### Шаг 2: Вычисление linelen

```
if (suffix > 0 || (just && (w2 != null || last)))
    linelen = L + affix
else if (w1 != null)
    linelen = prefix + L - extra
else
    linelen = prefix
```

где `affix = prefix + suffix`.

### Шаг 3: Копирование префикса

```
if (numout <= numIn)
    prefix_text = originalPrefixes[numout - 1]     // из исходной строки
else if (numIn > hang)
    prefix_text = originalPrefixes[numIn - 1]      // из последней исходной строки
else
    prefix_text = originalPrefixes[numIn - 1].Take(min(afp, prefix))
    // остальное — пробелы
```

### Шаг 4: Копирование тела (слов)

Для каждого слова на строке:
1. Копировать текст слова.
2. Если не последнее слово на строке:
   a. Добавить один пробел.
   b. Если `just` и (`w1.NextLine != null` или `last`):
      - Распределить дополнительные пробелы (phase algorithm).
   c. Если `w2.Shifted`: добавить ещё один пробел.

**Phase algorithm** (равномерное распределение):
```
phase = numgaps / 2
for each gap:
    phase += extra
    while (phase >= numgaps):
        add space
        phase -= numgaps
```

Это распределяет `extra` дополнительных пробелов равномерно по `numgaps` промежуткам.

### Шаг 5: Заполнение пробелами

```
while (current_pos < linelen - affix):
    add space
```

### Шаг 6: Копирование суффикса

Аналогично префиксу, но из `suffixes[]`.

## Адаптация для xpar

### Ключевые отличия

1. **Список слов** — `List<Word>` с `NextLine` как `int?`.
   - `w1 = head->next` → `wordIdx = 0` (первое слово)
   - `w1 = w1->nextline` → `wordIdx = words[wordIdx].NextLine`
   - `w2 != w1->nextline` → `j != words[wordIdx].NextLine`

2. **Длина** — визуальная ширина графем, не количество байт.

3. **Префиксы/суффиксы** — строки C#, не указатели на байты.
   - Нужно хранить оригинальные префиксы/суффиксы из `ExtractWordsResult.Suffixes`.
   - Префиксы нужно извлечь отдельно (они не хранятся в `ExtractWordsResult`).

4. **Результат** — `IReadOnlyList<string>`, не `char**`.

### Необходимые входные данные

Методу потребуется:
- `words` — список слов с расставленными NextLine.
- `suffixes` — суффиксы исходных строк (из `ExtractWordsResult`).
- `prefixes` — префиксы исходных строк (нужно добавить в `ExtractWordsResult` или передать отдельно).
- `L` — длина тела строки (может быть изменён при `touch`).
- `prefix`, `suffix` — длины.
- `hang` — минимальное количество строк.
- `just` — режим justification.
- `last` — учитывать последнюю строку.
- `touch` — изменить L на фактическую длину.
- `afp` — augmented fallback prelen.
- `fs` — fallback suflen.

### Сигнатура метода

```csharp
/// <summary>
/// Строит выходные строки абзаца из слов с расставленными разрывами.
/// </summary>
/// <remarks>Аналог блока "Construct the lines" из reformat.c (строки 459–522).</remarks>
public static IReadOnlyList<string> ConstructLines(
    List<Word> words,
    List<string> prefixes,
    List<string> suffixes,
    int L,
    SetAffixesResult affixes,
    int hang,
    bool just,
    bool last,
    bool touch)
```

где `affixes.Prefix`, `affixes.Suffix`, `affixes.AugmentedFallbackPre`, `affixes.FallbackSuf`.

### План реализации

```csharp
public static IReadOnlyList<string> ConstructLines(
    List<Word> words,
    List<string> prefixes,
    List<string> suffixes,
    int L,
    SetAffixesResult affixes,
    int hang,
    bool just,
    bool last,
    bool touch)
{
    int numIn = prefixes.Count;
    int prefix = affixes.Prefix;
    int suffix = affixes.Suffix;
    int affix = prefix + suffix;
    var result = new List<string>();

    // Фаза touch: изменить L на фактическую длину самой длинной строки
    if (!just && touch)
    {
        int maxLen = 0;
        int? wordIdx = words.Count > 0 ? 0 : null;
        while (wordIdx.HasValue)
        {
            int lineLen = words[wordIdx.Value].Width;
            int j = wordIdx.Value + 1;
            while (j < words.Count && j != words[wordIdx.Value].NextLine)
            {
                lineLen += 1 + (words[j].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0) + words[j].Width;
                j++;
            }
            if (lineLen > maxLen) maxLen = lineLen;
            wordIdx = words[wordIdx.Value].NextLine;
        }
        L = maxLen;
    }

    int numout = 0;
    int? wordIdx2 = words.Count > 0 ? 0 : null;

    while (numout < hang || wordIdx2.HasValue)
    {
        // Шаг 1: Вычисление numgaps и extra
        int numgaps = 0;
        int extra = L;
        if (wordIdx2.HasValue)
        {
            extra = L - words[wordIdx2.Value].Width;
            int j = wordIdx2.Value + 1;
            while (j < words.Count && j != words[wordIdx2.Value].NextLine)
            {
                numgaps++;
                extra -= 1 + (words[j].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0) + words[j].Width;
                j++;
            }
        }

        // Шаг 2: Вычисление linelen
        bool hasNextLine = wordIdx2.HasValue && words[wordIdx2.Value].NextLine.HasValue;
        int linelen;
        if (suffix > 0 || (just && (hasNextLine || last)))
            linelen = L + affix;
        else if (wordIdx2.HasValue)
            linelen = prefix + L - extra;
        else
            linelen = prefix;

        var sb = new StringBuilder(linelen);

        // Шаг 3: Копирование префикса
        string prefixText;
        if (numout < numIn)
            prefixText = prefixes[numout];
        else if (numIn > hang)
            prefixText = prefixes[numIn - 1];
        else
            prefixText = prefixes[numIn - 1];

        // Обрезать или дополнить префикс
        int prefixVisualWidth = CalcVisualWidth(prefixText);
        if (prefixVisualWidth >= prefix)
        {
            // Взять первые prefix графем
            var g = LineReader.GetGraphemes(prefixText);
            sb.Append(string.Concat(g.Take(prefix)));
        }
        else
        {
            sb.Append(prefixText);
            if (numout >= numIn && numIn <= hang)
            {
                // Дополнить пробелами до min(afp, prefix)
                int take = Math.Min(afp, prefix);
                for (int i = prefixVisualWidth; i < take; i++)
                    sb.Append(' ');
            }
        }

        // Шаг 4: Копирование тела (слов)
        int bodyStartPos = sb.Length;
        if (wordIdx2.HasValue)
        {
            int phase = numgaps / 2;
            int j = wordIdx2.Value;
            while (true)
            {
                sb.Append(words[j].Text);

                int nextJ = j + 1;
                if (nextJ >= words.Count || nextJ == words[wordIdx2.Value].NextLine)
                    break;

                sb.Append(' ');

                // Justification: распределить дополнительные пробелы
                if (just && (hasNextLine || last))
                {
                    phase += extra;
                    while (phase >= numgaps)
                    {
                        sb.Append(' ');
                        phase -= numgaps;
                    }
                }

                // Shifted: дополнительный пробел
                if (words[nextJ].Flags.HasFlag(WordFlags.Shifted))
                    sb.Append(' ');

                j = nextJ;
            }
        }

        // Шаг 5: Заполнение пробелами до linelen - affix
        int bodyEndPos = linelen - affix;
        int currentBodyWidth = CalcVisualWidth(sb.ToString().Substring(prefix));
        // Проще: дополнить пробелами до нужной позиции
        while (sb.Length < bodyEndPos)
            sb.Append(' ');

        // Шаг 6: Копирование суффикса
        string suffixText;
        if (numout < numIn)
            suffixText = suffixes[numout];
        else if (numIn > hang)
            suffixText = suffixes[numIn - 1];
        else
            suffixText = suffixes[numIn - 1];

        int suffixVisualWidth = CalcVisualWidth(suffixText);
        if (suffixVisualWidth >= suffix)
        {
            var g = LineReader.GetGraphemes(suffixText);
            int start = g.Count - suffix;
            sb.Append(string.Concat(g.Skip(start)));
        }
        else
        {
            if (numout >= numIn && numIn <= hang)
            {
                int take = Math.Min(fs, suffix);
                var g = LineReader.GetGraphemes(suffixText);
                int s = Math.Max(0, g.Count - take);
                sb.Append(string.Concat(g.Skip(s)));
                for (int i = suffixVisualWidth; i < suffix; i++)
                    sb.Append(' ');
            }
            else
            {
                sb.Append(suffixText);
            }
        }

        result.Add(sb.ToString());
        numout++;

        if (wordIdx2.HasValue)
            wordIdx2 = words[wordIdx2.Value].NextLine;
    }

    return result.AsReadOnly();
}
```

**Примечание:** В оригинале префиксы не хранятся отдельно — они извлекаются из исходных строк (`inlines[numout - 1]`). В xpar нужно либо:
1. Добавить `prefixes` в `ExtractWordsResult`.
2. Извлекать префиксы из исходных строк отдельно.

**Решение:** Добавить `Prefixes` в `ExtractWordsResult`.
