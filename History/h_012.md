# Шаг 12: Перенос simplebreaks — выбор разрывов строк (максимизация кратчайшей строки)

## Цель шага

Реализовать метод `SimpleBreaks` в `ReformatModule` — алгоритм выбора разрывов строк,
который максимизирует длину кратчайшей строки. Это вспомогательный метод,
используемый в `normalbreaks` (при `fit=1`) и являющийся основой для
выбора разрывов без justification.

## Контекст

В оригинале `simplebreaks` (reformat.c строки 98–136) — статическая функция,
которая:
- Принимает связный список слов (от `head` до `tail`)
- Выбирает разрывы строк так, чтобы **кратчайшая строка была как можно длиннее**
- Возвращает длину кратчайшей строки, `-1` если слово > L, или `L` если слов нет
- Заполняет `nextline` и `score` для каждого слова

Этот метод вызывается из `normalbreaks` (строки 138–196) для определения
оптимальной ширины при `fit=1`.

## Оригинал (reformat.c строки 98–136)

```c
static int simplebreaks(word *head, word *tail, int L, int last)
{
  word *w1, *w2;
  int linelen, score;

  if (!head->next) return L;

  // Фаза 1: инициализация хвоста (с конца к началу, пока linelen <= L)
  for (w1 = tail, linelen = w1->length;
       w1 != head && linelen <= L;
       linelen += isshifted(w1), w1 = w1->prev, linelen += 1 + w1->length) {
    w1->score = last ? linelen : L;
    w1->nextline = NULL;
  }

  // Фаза 2: основной DP (оставшиеся слова)
  for ( ;  w1 != head;  w1 = w1->prev) {
    w1->score = -1;
    for (linelen = w1->length,  w2 = w1->next;
         linelen <= L;
         linelen += 1 + isshifted(w2) + w2->length,  w2 = w2->next) {
      score = w2->score;
      if (linelen < score) score = linelen;
      if (score >= w1->score) {
        w1->nextline = w2;
        w1->score = score;
      }
    }
  }

  return head->next->score;
}
```

## Алгоритм (пошагово)

### Фаза 1: Инициализация хвоста

Идём от последнего слова (`tail`) назад к `head`, накапливая длину строки:
- `linelen = w1->length` (длина текущего слова)
- Пока `linelen <= L`:
  - `w1->score = last ? linelen : L` — если `last=1`, score = длина строки; иначе = L
  - `w1->nextline = NULL` — нет следующей строки (все оставшиеся слова помещаются)
  - Переходим к предыдущему: `linelen += isshifted(w1) + 1 + w1->prev->length`

**Смысл:** если все оставшиеся слова помещаются на одну строку,
то `nextline = NULL` (нет разрыва), а score зависит от `last`.

### Фаза 2: Основной DP

Для оставшихся слов (которые не поместились в фазе 1):
- `w1->score = -1` (по умолчанию — невозможно)
- Для каждого возможного конца строки `w2` (от `w1->next` и далее, пока `linelen <= L`):
  - `score = min(linelen, w2->score)` — кратчайшая строка от w1 до конца
  - Если `score >= w1->score` (лучше или равно текущему):
    - `w1->nextline = w2` — разрыв перед w2
    - `w1->score = score`

**Смысл:** для каждого слова находим такой разрыв, чтобы минимальная
длина строки от этого слова до конца была максимальной.

### Возвращаемое значение

- `head->next->score` — длина кратчайшей строки всего абзаца
- `-1` — если есть слово шире L (невозможно)
- `L` — если слов нет

## Адаптация для xpar

### Ключевые отличия

1. **Список слов** — `List<Word>`, а не связный список.
   - `w1->prev` → `words[idx - 1]`
   - `w1->next` → `words[idx + 1]`
   - `w1->nextline` → `w1.NextLine` (int? — индекс слова)

2. **Длина строки** — визуальная ширина, включая `Shifted` и пробелы:
   - `linelen += isshifted(w)` → `linelen += (w.Flags.HasFlag(WordFlags.Shifted) ? 1 : 0)`
   - `linelen += 1 + w2->length` → `linelen += 1 + w2.Width`

3. **Итерация назад** — в оригинале `w1 = w1->prev`. В xpar: `idx--`.

### Сигнатура метода

```csharp

/// </summary>
```

### План реализации

```csharp
private static int SimpleBreaks(List<Word> words, int L, bool last)
{
    if (words.Count == 0) return L;

    int n = words.Count;

    // Фаза 1: инициализация хвоста (с конца к началу)
    int idx = n - 1;
    int linelen = words[idx].Width;

    while (idx >= 0 && linelen <= L)
    {
        words[idx].Score = last ? linelen : L;
        words[idx].NextLine = null;

        // Переход к предыдущему слову
        idx--;
        if (idx < 0) break;

        linelen += (words[idx + 1].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0);
        linelen += 1 + words[idx].Width;
    }

    // Фаза 2: основной DP
    while (idx >= 0)
    {
        words[idx].Score = -1;

        linelen = words[idx].Width;
        int j = idx + 1;

        while (j < n && linelen <= L)
        {
            int score = words[j].Score;
            if (linelen < score) score = linelen;

            if (score >= words[idx].Score)
            {
                words[idx].NextLine = j;
                words[idx].Score = score;
            }

            linelen += 1 + (words[j].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0) + words[j].Width;
            j++;
        }

        idx--;
    }

    return words[0].Score;
}
```
