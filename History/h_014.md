# Шаг 14: Перенос justbreaks — выбор разрывов строк (justification)

## Цель шага

Реализовать метод `JustBreaks` в `ReformatModule` — алгоритм выбора разрывов строк
для режима justification (`just=1`). Этот метод обеспечивает выравнивание текста
по обоим краям, равномерно распределяя дополнительные пробелы между словами.

## Контекст

В оригинале `justbreaks` (reformat.c строки 211–289) — статическая функция,
которая выбирает разрывы строк согласно политике из par.doc для `just=1`:

1. **Фаза 1:** определить минимальный возможный максимальный межсловный промежуток (`maxgap`).
2. **Фаза 2:** минимизировать сумму квадратов количеств дополнительных пробелов
   в каждом межсловном промежутке.

Этот метод вызывается из `reformat()` (строка 435):
```c
if (just) justbreaks(head,tail,L,last,errmsg);
else normalbreaks(head,tail,L,fit,last,errmsg);
```

## Оригинал (reformat.c строки 211–289)

```c
static void justbreaks(
  word *head, word *tail, int L, int last, errmsg_t errmsg
)
{
  word *w1, *w2;
  int numgaps, extra, score, gap, maxgap, numbiggaps;

  *errmsg = '\0';
  if (!head->next) return;

  /* Фаза 1: Determine the minimum possible largest inter-word gap */
  w1 = tail;
  do {
    w1->score = L;
    for (numgaps = 0, extra = L - w1->length, w2 = w1->next;
         extra >= 0;
         ++numgaps, extra -= 1 + isshifted(w2) + w2->length, w2 = w2->next) {
      gap = numgaps ? (extra + numgaps - 1) / numgaps : L;
      if (w2)
        score = w2->score;
      else {
        score = 0;
        if (!last) gap = 0;
      }
      if (gap > score) score = gap;
      if (score < w1->score) {
        w1->nextline = w2;
        w1->score = score;
      }
      if (!w2) break;
    }
    w1 = w1->prev;
  } while (w1 != head);

  maxgap = head->next->score;
  if (maxgap >= L) {
    strcpy(errmsg, "Cannot justify.\n");
    return;
  }

  /* Фаза 2: Minimize the sum of the squares of the numbers */
  /* of extra spaces required in each inter-word gap: */
  w1 = tail;
  do {
    w1->score = -1;
    for (numgaps = 0, extra = L - w1->length, w2 = w1->next;
         extra >= 0;
         ++numgaps, extra -= 1 + isshifted(w2) + w2->length, w2 = w2->next) {
      gap = numgaps ? (extra + numgaps - 1) / numgaps : L;
      if (w2)
        score = w2->score;
      else {
        if (!last) {
          w1->nextline = NULL;
          w1->score = 0;
          break;
        }
        score = 0;
      }
      if (gap <= maxgap && score >= 0) {
        numbiggaps = extra % numgaps;
        score += (extra / numgaps) * (extra + numbiggaps) + numbiggaps;
        if (w1->score < 0  ||  score <= w1->score) {
          w1->nextline = w2;
          w1->score = score;
        }
      }
      if (!w2) break;
    }
    w1 = w1->prev;
  } while (w1 != head);

  if (head->next->score < 0)
    sprintf(errmsg,impossibility,3);
}
```

## Алгоритм (пошагово)

### Фаза 1: Минимизация максимального межсловного промежутка

Для каждого слова `w1` (от tail к head):
- `w1->score = L` (начальное значение — худший случай)
- `extra = L - w1->length` — свободное место в строке
- `numgaps = 0` — количество межсловных промежутков
- Для каждого кандидата `w2` (конец строки):
  - `gap = numgaps > 0 ? ceil(extra / numgaps) : L` — средний размер промежутка
    - `(extra + numgaps - 1) / numgaps` — целочисленное округление вверх
  - `score = max(gap, w2->score)` — максимальный промежуток от w1 до конца
  - Если `score < w1->score` (лучше):
    - `w1->nextline = w2`
    - `w1->score = score`

**Результат:** `maxgap = words[0].Score` — минимальный возможный максимальный промежуток.

**Проверка:** если `maxgap >= L` — невозможно выровнять (ошибка "Cannot justify").

### Фаза 2: Минимизация суммы квадратов дополнительных пробелов

Для каждого слова `w1` (от tail к head):
- `w1->score = -1` (по умолчанию — невозможно)
- `extra = L - w1->length`, `numgaps = 0`
- Для каждого кандидата `w2`:
  - `gap = numgaps > 0 ? ceil(extra / numgaps) : L`
  - Если `w2 == null` (последняя строка):
    - Если `!last`: `nextline = null`, `score = 0`, break
    - Иначе: `score = 0`
  - Иначе: `score = w2->score`
  - Если `gap <= maxgap && score >= 0`:
    - `numbiggaps = extra % numgaps` — количество промежутков с доп. пробелом
    - `score += (extra / numgaps) * (extra + numbiggaps) + numbiggaps`
      - Это **сумма квадратов** количеств дополнительных пробелов в каждом промежутке
      - Формула: если `extra = q * numgaps + r`, то `r` промежутков имеют `q+1` пробелов,
        `numgaps - r` имеют `q` пробелов.
        Сумма квадратов: `r*(q+1)² + (numgaps-r)*q² = q*(q*numgaps + 2r) + r = q*(extra+r) + r`
        = `(extra/numgaps) * (extra + numbiggaps) + numbiggaps`
    - Если `score <= w1->score` (лучше или равно):
      - `w1->nextline = w2`
      - `w1->score = score`

**Результат:** оптимальные разрывы строк для justification.

### Проверка ошибки

Если `head->next->score < 0` — ни один вариант не подошёл, ошибка "impossibility 3".

## Адаптация для xpar

### Ключевые отличия

1. **Список слов** — `List<Word>`, а не связный список.
   - `w1->prev` → `words[idx - 1]`
   - `w1->next` → `words[j]`
   - `w1->nextline` → `w1.NextLine` (int?)
   - `head->next` → `words[0]`

2. **Длина строки** — визуальная ширина:
   - `extra = L - w1->length` → `extra = L - words[idx].Width`
   - `extra -= 1 + isshifted(w2) + w2->length`
   - → `extra -= 1 + (w2.Flags.Shifted ? 1 : 0) + w2.Width`

3. **Цикл `do-while`** — в оригинале `do { ... } while (w1 != head)`.
   В xpar: `for (idx = n-1; idx >= 0; idx--)`.

### План реализации

```csharp
public static string? JustBreaks(List<Word> words, int L, bool last)
{
    if (words.Count == 0) return null;

    int n = words.Count;

    // === Фаза 1: Минимизация максимального межсловного промежутка ===
    for (int idx = n - 1; idx >= 0; idx--)
    {
        words[idx].Score = L;

        int extra = L - words[idx].Width;
        int numgaps = 0;
        int j = idx + 1;

        while (j <= n && extra >= 0)
        {
            int gap = numgaps > 0 ? (extra + numgaps - 1) / numgaps : L;
            int score = (j < n) ? words[j].Score : 0;

            if (j >= n && !last)
                gap = 0;

            if (gap > score) score = gap;

            if (score < words[idx].Score)
            {
                words[idx].NextLine = (j < n) ? j : null;
                words[idx].Score = score;
            }

            if (j >= n) break;

            extra -= 1 + (words[j].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0) + words[j].Width;
            numgaps++;
            j++;
        }
    }

    int maxgap = words[0].Score;
    if (maxgap >= L)
        return "Cannot justify.";

    // === Фаза 2: Минимизация суммы квадратов дополнительных пробелов ===
    for (int idx = n - 1; idx >= 0; idx--)
    {
        words[idx].Score = -1;

        int extra = L - words[idx].Width;
        int numgaps = 0;
        int j = idx + 1;

        while (j <= n && extra >= 0)
        {
            int gap = numgaps > 0 ? (extra + numgaps - 1) / numgaps : L;

            if (j >= n)
            {
                if (!last)
                {
                    words[idx].NextLine = null;
                    words[idx].Score = 0;
                    break;
                }
                // last=true: score = 0
            }

            int score = (j < n) ? words[j].Score : 0;

            if (gap <= maxgap && score >= 0)
            {
                int numbiggaps = (numgaps > 0) ? extra % numgaps : 0;
                score += (extra / numgaps) * (extra + numbiggaps) + numbiggaps;

                if (words[idx].Score < 0 || score <= words[idx].Score)
                {
                    words[idx].NextLine = (j < n) ? j : null;
                    words[idx].Score = score;
                }
            }

            if (j >= n) break;

            extra -= 1 + (words[j].Flags.HasFlag(WordFlags.Shifted) ? 1 : 0) + words[j].Width;
            numgaps++;
            j++;
        }
    }

    if (words[0].Score < 0)
        return "Cannot format paragraph (impossibility 3)";

    return null;
}
```
