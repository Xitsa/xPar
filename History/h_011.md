# Шаг 11: Выделение списка слов, обработка guess и проверка длины

## Цель шага

Реализовать в `ReformatModule` метод `ExtractWords`, который объединяет три этапа
оригинальной функции `reformat()`:

1. **Выделение слов** из строк абзаца (reformat.c строки 331–368)
2. **Обработка guess** — пометка curious/capital слов и слияние (reformat.c строки 371–397)
3. **Проверка/разбиение длинных слов** — если слово > L, разбить или выдать ошибку (reformat.c строки 400–430)

## Контекст

Функция `reformat()` из `reformat.c` выполняет несколько последовательных этапов:

1. **Инициализация** — проверка входных данных, выделение памяти
2. **Выделение слов** — создание связного списка `word` из строк абзаца (строки 331–368)
3. **Обработка guess** — пометка curious/capital слов и слияние (строки 371–397)
4. **Проверка/разбиение длинных слов** — если слово > L, разбить или выдать ошибку (строки 400–430)
5. **Выбор разрывов строк** — `justbreaks` или `normalbreaks` (строки 433–436)
6. **Построение выходных строк** — сборка отформатированных строк (строки 439–502)

**На этом шаге реализуются этапы 2, 3 и 4 — объединённые в метод `ExtractWords`.**

## Структура `word` (аналог `struct word` из reformat.c)

В оригинале (reformat.c строки 31–44):

```c
typedef struct word {
  const char *chrs;       /* Pointer to the characters in the word */
                          /* (NOT terminated by '\0').             */
  struct word *prev,      /* Pointer to previous word.             */
              *next,      /* Pointer to next word.                 */
                          /* Supposing this word were the first... */
              *nextline;  /*   Pointer to first word in next line. */
  int score,              /*   Value of the objective function.    */
      length;             /* Length of this word.                  */
  wflag_t flags;          /* Notable properties of this word.      */
} word;
```

### Соответствие полей

| Поле в оригинале (C) | Тип C | Поле в xpar (C#) | Тип C# | Описание |
|---|---|---|---|---|
| `chrs` | `const char *` | `Text` | `string` | Текст слова (в оригинале — указатель без `\0`, в C# — полная строка) |
| `prev` | `word *` | — | — | Не нужен: слова хранятся в `List<Word>`, предыдущее — по индексу |
| `next` | `word *` | — | — | Не нужен: слова хранятся в `List<Word>`, следующее — по индексу |
| `nextline` | `word *` | `NextLine` | `int?` | Индекс первого слова следующей строки (для word wrapping) |
| `score` | `int` | `Score` | `int` | Значение целевой функции (для алгоритма разбиения) |
| `length` | `int` | `Width` | `int` | Визуальная ширина слова в графемах (не количество символов!) |
| `flags` | `wflag_t` | `Flags` | `WordFlags` | Флаги свойств слова |

**Почему `List<Word>` вместо связного списка:** В оригинале `word` — это struct с указателями.
В C# удобнее хранить слова в сплошном списке (`List<Word>`), а `NextLine` — как индекс.
Это упрощает итерацию, разбиение длинных слов и будущий word wrapping.

## Алгоритм выделения слов

### Оригинал (reformat.c строки 331–368)

```c
affix = prefix + suffix;
L = width - prefix - suffix;

line = inlines, suf = suffixes;
do {
    for (end = *line;  *end;  ++end);
    if (end - *line < affix) {
        sprintf(errmsg, "Line %ld shorter than <prefix> + <suffix> = %d + %d = %d\n",
                (long)(line - inlines + 1), prefix, suffix, affix);
        goto rfcleanup;
    }
    end -= suffix;
    *suf = end;
    p1 = *line + prefix;
    for (;;) {
        while (p1 < end && *p1 == ' ') ++p1;
        if (p1 == end) break;
        p2 = p1;
        if (onfirstword) {
            p1 = *line + prefix;
            onfirstword = 0;
        }
        while (p2 < end && *p2 != ' ') ++p2;
        w1 = malloc(sizeof (word));
        // ... инициализация w1 ...
        w1->chrs = p1;
        w1->length = p2 - p1;
        w1->flags = 0;
        p1 = p2;
    }
    ++line, ++suf;
} while (line < endline);
```

### Логика алгоритма (пошагово)

1. **Вычисление L** — максимальная длина тела строки: `L = width - prefix - suffix`
2. **Для каждой строки** абзаца:
   a. Проверить, что строка достаточно длинная: `длина >= prefix + suffix`
   b. Сохранить указатель на суффикс (для последующего копирования)
   c. Начать разбор тела с позиции `prefix` до `длина - suffix`
3. **Для каждого слова в теле**:
   a. Пропустить ведущие пробелы
   b. Если достигнут конец тела — перейти к следующей строке
   c. **Специальное правило для первого слова** (строки 350–353): `p1` сбрасывается к началу тела
      (это нужно, чтобы первое слово включало всё тело от начала, даже если были пробелы)
   d. Найти конец слова (до следующего пробела или конца тела)
   e. Создать `word` с `chrs = p1`, `length = p2 - p1`
   f. Продолжить с `p1 = p2`

### Адаптация для Unicode (xpar)

Ключевые отличия:

1. **Длина слова** — визуальная ширина в графемах (через `Wcwidth.UnicodeCalculator.GetWidth()`),
   а не количество байт/символов.
2. **Пробелы** — проверяем `grapheme == " "`, так как все whitespace-символы уже унифицированы
   в `LineReader.ProcessLine` (строка 192: `whiteChars.IsMember(grapheme)` → `" "`).
3. **Строки** — работаем с `string` и `StringInfo.GetTextElementEnumerator` для графем.
4. **Проверка длины строки** — сравниваем визуальную ширину префикса + суффикса с `width`.

### Алгоритм обработки guess (reformat.c строки 371–397)

Оригинал:

```c
if (guess) {
    for (w1 = head, w2 = head->next;  w2;  w1 = w2, w2 = w2->next) {
        if (checkcurious(w2,terminalchars)) w2->flags |= W_CURIOUS;
        if (cap || checkcapital(w2)) {
            w2->flags |= W_CAPITAL;
            if (iscurious(w1)) {
                if (w1->chrs[w1->length] && w1->chrs + w1->length + 1 == w2->chrs) {
                    w2->length += w1->length + 1;
                    w2->chrs = w1->chrs;
                    w2->prev = w1->prev;
                    w2->prev->next = w2;
                    if (iscapital(w1)) w2->flags |= W_CAPITAL;
                    else w2->flags &= ~W_CAPITAL;
                    if (isshifted(w1)) w2->flags |= W_SHIFTED;
                    else w2->flags &= ~W_SHIFTED;
                    free(w1);
                }
                else w2->flags |= W_SHIFTED;
            }
        }
    }
    tail = w1;
}
```

**Логика:**

1. Для каждого слова `w2` (итерация по списку):
   a. Проверить, является ли `w2` **curious** — содержит terminal-символ, после которого нет alphanumeric, но есть alphanumeric до него.
   b. Если `cap == 1` или слово **capitalized** — пометить `W_CAPITAL`.
   c. Если предыдущее слово `w1` — curious, а текущее `w2` — capitalized:
      - Если между ними **ровно один пробел** (в оригинале: `w1->chrs[w1->length]` существует и `w1->chrs + w1->length + 1 == w2->chrs`):
        - **Слить** `w1` и `w2` в одно слово: `w2` получает текст `w1`, длину `w1.length + 1 + w2.length`, флаги объединяются.
        - Удалить `w1` из списка.
      - Иначе (разделены более чем одним пробелом или разрывом строки):
        - Пометить `w2` как `W_SHIFTED` (дополнительный пробел перед словом).

**Определения (par.doc):**

- **capitalized word**: слово, содержащее хотя бы один alphanumeric-символ, первый alphanumeric — не lowercase. Если `cap == 1`, все слова считаются capitalized.
- **curious word**: слово, содержащее terminal-символ `c`, такой что после `c` нет alphanumeric, но хотя бы один alphanumeric есть до `c`.

**Адаптация для Unicode:**

- Проверка `w1->chrs[w1->length]` (символ после конца слова) — в xpar нужно хранить информацию о разделителе между словами.
- Проверка смежности слов: в оригинале указатели смежны (`w1->chrs + w1->length + 1 == w2->chrs`). В xpar — нужно хранить индекс строки и позицию.

**Решение:** добавить в `Word` поле `SourceLineIndex` (индекс строки в segments) и `SourceEndSeparator` (символ-разделитель после слова, если есть).

### Алгоритм проверки/разбиения длинных слов (reformat.c строки 400–430)

Оригинал:

```c
if (Report)
    for (w2 = head->next;  w2;  w2 = w2->next) {
        if (w2->length > L) {
            linelen = w2->length;
            if (linelen > errmsg_size - 17)
                linelen = errmsg_size - 17;
            sprintf(errmsg, "Word too long: %.*s\n", linelen, w2->chrs);
            goto rfcleanup;
        }
    }
else
    for (w2 = head->next;  w2;  w2 = w2->next)
        while (w2->length > L) {
            w1 = malloc(sizeof (word));
            // ... вставка w1 перед w2 ...
            w1->chrs = w2->chrs;
            w2->chrs += L;
            w1->length = L;
            w2->length -= L;
            w1->flags = 0;
            if (iscapital(w2)) {
                w1->flags |= W_CAPITAL;
                w2->flags &= ~W_CAPITAL;
            }
            if (isshifted(w2)) {
                w1->flags |= W_SHIFTED;
                w2->flags &= ~W_SHIFTED;
            }
        }
```

**Логика:**

1. Если `Report == 1`: проверить все слова, если есть слово > L — ошибка "Word too long".
2. Если `Report == 0`: разбить каждое слово > L на части длиной L:
   - Создать новое слово перед текущим.
   - Первое слово получает первые L символов, второе — остаток.
   - Флаги `Capital` и `Shifted` переносятся на первое слово, второе — без флагов.

**Адаптация для Unicode:**

- Разбиение по графемам: нужно разбить так, чтобы каждая часть имела визуальную ширину ≤ L.
- Поскольку L — это максимальная визуальная ширина, разбиваем по символам (не графемам), пока суммарная ширина ≤ L.

### План реализации метода `ExtractWords`

```csharp
/// <summary>
/// Выделяет список слов из сегментов абзаца, обрабатывает guess и проверяет длину.
/// </summary>
/// <remarks>Аналог блоков выделения слов, guess и проверки длины из reformat() (reformat.c строки 331–430).</remarks>
private static ExtractWordsResult ExtractWords(
    IReadOnlyList<LineSegment> segments,
    int startIndex,
    int endIndex,
    int prefix,
    int suffix,
    int width,
    Charset terminalChars,
    bool cap,
    bool guess,
    bool report)
{
    int L = width - prefix - suffix;
    int numIn = endIndex - startIndex + 1;
    var suffixes = new List<string>(numIn);

    var words = new List<Word>();
    bool onFirstWord = true;

    // === Этап 1: Выделение слов (reformat.c 331–368) ===
    for (int i = startIndex; i <= endIndex; i++)
    {
        string line = segments[i].Line;
        int lineVisualWidth = CalcVisualWidth(line);
        int affix = prefix + suffix;

        if (lineVisualWidth < affix)
        {
            return new ExtractWordsResult(
                words: new List<Word>(),
                suffixes: new List<string>(),
                L: 0,
                errorMessage: $"Line {i - startIndex + 1} shorter than <prefix> + <suffix> = {prefix} + {suffix} = {affix}");
        }

        suffixes.Add(GetSuffix(line, suffix));

        var graphemes = GetGraphemes(line);
        int bodyStart = prefix;
        int bodyEnd = lineVisualWidth - suffix;

        int p1 = bodyStart;
        while (p1 < bodyEnd)
        {
            // Пропустить пробелы (уже унифицированы в LineReader)
            while (p1 < bodyEnd && graphemes[p1] == " ")
                p1++;
            if (p1 >= bodyEnd) break;

            if (onFirstWord)
            {
                p1 = bodyStart;
                onFirstWord = false;
            }

            // Найти конец слова (до следующего пробела)
            int p2 = p1;
            while (p2 < bodyEnd && graphemes[p2] != " ")
                p2++;

            string wordText = string.Concat(graphemes.Skip(p1).Take(p2 - p1));
            int wordWidth = CalcGraphemeWidthRange(graphemes, p1, p2);

            // Определить разделитель после слова
            string separator = (p2 < bodyEnd && graphemes[p2] == " ") ? " " : "";

            words.Add(new Word
            {
                Text = wordText,
                Width = wordWidth,
                Flags = WordFlags.None,
                SourceLineIndex = i - startIndex,
                SeparatorAfterWord = separator,
            });

            p1 = p2;
        }
    }

    // === Этап 2: Обработка guess (reformat.c 371–397) ===
    if (guess)
    {
        for (int idx = 0; idx < words.Count; idx++)
        {
            var w2 = words[idx];
            if (IsCurious(w2, terminalChars))
                w2.Flags |= WordFlags.Curious;

            if (cap || IsCapitalized(w2))
            {
                w2.Flags |= WordFlags.Capital;

                if (idx > 0)
                {
                    var w1 = words[idx - 1];
                    if (w1.Flags.HasFlag(WordFlags.Curious))
                    {
                        // Проверка: разделены ли одним пробелом
                        if (w1.SourceLineIndex == w2.SourceLineIndex &&
                            w1.SeparatorAfterWord == " ")
                        {
                            // Слияние: w2 поглощает w1
                            w2.Text = w1.Text + " " + w2.Text;
                            w2.Width = w1.Width + 1 + w2.Width;
                            // Флаги: capital — если оба capital, иначе снять
                            if (!w1.Flags.HasFlag(WordFlags.Capital))
                                w2.Flags &= ~WordFlags.Capital;
                            if (w1.Flags.HasFlag(WordFlags.Shifted))
                                w2.Flags |= WordFlags.Shifted;
                            // Удалить w1 из списка
                            words.RemoveAt(idx - 1);
                            idx--; // скорректировать индекс
                        }
                        else
                        {
                            w2.Flags |= WordFlags.Shifted;
                        }
                    }
                }
            }
            words[idx] = w2;
        }
    }

    // === Этап 3: Проверка/разбиение длинных слов (reformat.c 400–430) ===
    if (report)
    {
        foreach (var w in words)
        {
            if (w.Width > L)
            {
                return new ExtractWordsResult(
                    words: new List<Word>(),
                    suffixes: new List<string>(),
                    L: 0,
                    errorMessage: $"Word too long: {w.Text}");
            }
        }
    }
    else
    {
        int idx = 0;
        while (idx < words.Count)
        {
            var w = words[idx];
            if (w.Width <= L)
            {
                idx++;
                continue;
            }

            // Разбить слово на части
            var (firstPart, secondPart) = SplitWord(w, L);

            // firstPart вставляется перед w, w становится secondPart
            words[idx] = firstPart;
            words.Insert(idx + 1, secondPart);

            idx++; // перейти к secondPart (проверить, нужно ли ещё разбить)
        }
    }

    return new ExtractWordsResult(
        words: words,
        suffixes: suffixes,
        L: L,
        errorMessage: null);
}
```

## Вспомогательные методы

### `LineReader.GetGraphemes(string)` — список графем строки

Метод `GetGraphemes` в `LineReader` делается публичным (был private).
Используется из `ReformatModule` без дублирования.

```csharp
// LineReader.cs (был private, стал public)
public static IReadOnlyList<string> GetGraphemes(string line)
{
    if (string.IsNullOrEmpty(line))
        return Array.Empty<string>();

    var list = new List<string>();
    var enumerator = StringInfo.GetTextElementEnumerator(line);
    while (enumerator.MoveNext())
        list.Add(enumerator.GetTextElement());
    return list.AsReadOnly();
}
```

### `CalcVisualWidth(string)` — визуальная ширина всей строки

```csharp
private static int CalcVisualWidth(string line)
{
    int width = 0;
    var enumerator = StringInfo.GetTextElementEnumerator(line);
    while (enumerator.MoveNext())
        width += Wcwidth.UnicodeCalculator.GetWidth(enumerator.GetTextElement()[0]);
    return width;
}
```

### `GetSuffix(string line, int suffixCount)` — извлечение суффикса

```csharp
private static string GetSuffix(string line, int suffixCount)
{
    if (suffixCount == 0) return "";
    var graphemes = GetGraphemes(line);
    int start = graphemes.Count - suffixCount;
    return string.Concat(graphemes.Skip(start));
}
```

### `CalcGraphemeWidthRange(IReadOnlyList<string> graphemes, int from, int to)` — ширина диапазона графем

```csharp
private static int CalcGraphemeWidthRange(IReadOnlyList<string> graphemes, int from, int to)
{
    int width = 0;
    for (int i = from; i < to; i++)
        width += Wcwidth.UnicodeCalculator.GetWidth(graphemes[i][0]);
    return width;
}
```

### `IsCurious(Word word, Charset terminalChars)` — проверка на curious word

```csharp
/// <summary>
/// Проверяет, является ли слово «любопытным» (curious).
/// Curious word — слово, содержащее terminal-символ, после которого нет alphanumeric,
/// но хотя бы один alphanumeric есть до него.
/// </summary>
/// <remarks>Аналог checkcurious() из reformat.c (строки 73–92).</remarks>
private static bool IsCurious(Word word, Charset terminalChars)
{
    var graphemes = GetGraphemes(word.Text);
    bool foundAlnumBefore = false;
    bool foundTerminal = false;

    // Идём с конца: ищем terminal, после которого нет alphanumeric
    for (int i = graphemes.Count - 1; i >= 0; i--)
    {
        string g = graphemes[i];
        if (IsAlphanumeric(g))
        {
            if (foundTerminal)
                return true; // alphanumeric до terminal
            foundAlnumBefore = true;
        }
        else if (terminalChars.IsMember(g))
        {
            foundTerminal = true;
        }
    }

    return false;
}
```

### `IsCapitalized(Word word)` — проверка на capitalized word

```csharp
/// <summary>
/// Проверяет, является ли слово капитализированным.
/// Capitalized word — слово с хотя бы одним alphanumeric, первый alphanumeric — не lowercase.
/// </summary>
/// <remarks>Аналог checkcapital() из reformat.c (строки 58–68).</remarks>
private static bool IsCapitalized(Word word)
{
    var graphemes = GetGraphemes(word.Text);
    foreach (var g in graphemes)
    {
        if (IsAlphanumeric(g))
        {
            // Первый alphanumeric — проверяем, не lowercase ли он
            char ch = g[0];
            return !char.IsLower(ch);
        }
    }
    return false; // нет alphanumeric
}
```

### `IsAlphanumeric(string grapheme)` — проверка на alphanumeric

```csharp
private static bool IsAlphanumeric(string grapheme)
{
    if (string.IsNullOrEmpty(grapheme)) return false;
    char ch = grapheme[0];
    return char.IsLetterOrDigit(ch);
}
```

### `SplitWord(Word word, int maxLength)` — разбиение длинного слова

```csharp
/// <summary>
/// Разбивает слово на две части так, что первая часть имеет визуальную ширину maxLength.
/// Возвращает два новых Word (first, second).
/// </summary>
private static (Word first, Word second) SplitWord(Word word, int maxLength)
{
    var graphemes = GetGraphemes(word.Text);
    int width = 0;
    int splitAt = 0;

    for (int i = 0; i < graphemes.Count; i++)
    {
        int w = Wcwidth.UnicodeCalculator.GetWidth(graphemes[i][0]);
        if (width + w > maxLength)
            break;
        width += w;
        splitAt = i + 1;
    }

    string firstText = string.Concat(graphemes.Take(splitAt));
    string secondText = string.Concat(graphemes.Skip(splitAt));

    var first = new Word
    {
        Text = firstText,
        Width = width,
        Flags = WordFlags.None,
        SourceLineIndex = word.SourceLineIndex,
        SeparatorAfterWord = "",
    };

    var second = new Word
    {
        Text = secondText,
        Width = CalcGraphemeWidthRange(graphemes, splitAt, graphemes.Count),
        Flags = word.Flags,
        SourceLineIndex = word.SourceLineIndex,
        SeparatorAfterWord = word.SeparatorAfterWord,
    };

    // Capital и Shifted переносятся на first, second — без этих флагов
    if (word.Flags.HasFlag(WordFlags.Capital))
    {
        first.Flags |= WordFlags.Capital;
        second.Flags &= ~WordFlags.Capital;
    }
    if (word.Flags.HasFlag(WordFlags.Shifted))
    {
        first.Flags |= WordFlags.Shifted;
        second.Flags &= ~WordFlags.Shifted;
    }

    return (first, second);
}
```

### Отличия от плана

- `bodyEnd` вычисляется через `graphemes.Count` (количество графем), а не через визуальную ширину — префикс/суффикс задаются в графемах.

- `SeparatorAfterWord` хранит все пробелы между словами (не только один), что нужно для guess-логики (один vs два пробела).
